## 游戏规则

1. 使用 `W`、`A`、`S`、`D` 控制飞船，按下 `Space`发射。
2. 燃料初始值为15，每秒递减1点，当燃料值为0时，游戏结束。
3. 触碰掉下的燃料瓶可增加15点，最大值为30点。
4. 击中敌方飞船增加5分。行星需击中两次，增加10分。击中友方扣除10分。
5. 碰撞敌方损失15点燃料，碰撞友方扣除10分。
6. 游戏允许负分。
7. 按下 `P` 暂停游戏，按下 `M` 静音。
